# 🏥 CarePulse AI

## 💙 Emergency Department Health Assistant

CarePulse AI is a Retrieval-Augmented Generation (RAG) chatbot designed to provide educational insights about Emergency Department (ER) operations.

---

## ✨ Features

- 🚑 Emergency Department (ER) information
- 📋 Triage level explanations
- ⏳ Patient wait time insights
- 📈 ER demand and overcrowding information
- 🌦️ Weather impact on ER visits
- 📅 Holiday impact on ER demand
- 🏥 Hospital workflow guidance
- 🤖 Retrieval-Augmented Generation (RAG)
- 💬 Interactive healthcare chatbot interface

---

## 🛠️ Technologies Used

- 🐍 Python
- 🎈 Streamlit
- 🦙 LlamaIndex
- ⚡ Groq LLM
- 🤗 HuggingFace Embeddings
- 🔍 Sentence Transformers
- 🧠 Retrieval-Augmented Generation (RAG)

## 🚀 How to Run

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/carepulse-ai-chatbot.git
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Add your Groq API key

```toml
GROQ_API_KEY="YOUR_API_KEY"
```

### 4️⃣ Run the application

```bash
streamlit run app.py
```
